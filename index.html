ターミナルで cd ダウンロードフォルダ → python3 -m http.server 8080
ブラウザで http://localhost:8080/chaos_map_v2.html を開く
🔧 Google Cloud セットアップ（3分）
① Google Cloud Console → プロジェクト選択（なければ新規作成）
② 「認証情報」→「＋認証情報を作成」→「OAuth クライアント ID」
③ アプリの種類：ウェブ アプリケーション
④ 「承認済みの JavaScript 生成元」に追加：
    http://localhost:8080
⑤ 「APIとサービス」→「ライブラリ」→ Google Slides API を有効化
